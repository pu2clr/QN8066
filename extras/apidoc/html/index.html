QN8066 Arduino Library implementation

This is an Arduino library for the QN8066 FM RX/TX device (Digital FM Transceiver for Portable Devices).

The communication used by this library is I2C.

You can see a complete documentation on https://github.com/pu2clr/QN8066 or https://pu2clr.github.io/QN8066/

This library can be freely distributed. This means that you can copy, modify, merge, and redistribute the code for free under the terms of the MIT License.

There are examples that can help you in your project on https://github.com/pu2clr/QN8066/tree/master/examples
